# netflix-streamlit

## Netflix Dashboard (Streamlit)

This project is an interactive dashboard built using Streamlit to analyze Netflix content data.

## Features

- Content growth over years (Movies vs TV Shows)
- Top 10 genres
- Top 10 directors
- Top 10 countries producing content
- Country-based filtering
- Title-based detailed view
- Global filter for Movies / TV Shows

## Tech Stack

- Python
- Pandas
- Matplotlib
- Seaborn
- Streamlit

##  Dataset

Netflix dataset containing information about:
- Title
- Type (Movie / TV Show)
- Genre
- Director
- Country
- Release Year

## How to Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
